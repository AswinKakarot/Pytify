PythonSpotify
=============

Search and start songs from command line.<br>

### Usage
```python
# To start the app run this
./cli.py
```

For quick link to this app add it to usr local bin. Ex:<br>
sudo ln -s ~/Path/To/cli.py  /usr/local/bin/sptfy <br>

#### Usage is now
```python
# To start the app run this
sptfy
```
